# SelOO 功能简介

SelOO 是一个基于 **OpenSceneGraph (OSG)** 和 **OpenCascade (OCC)** 开发的三维模型查看器，主要用于 CAD 模型的加载、显示与交互操作。它在轻量级架构下提供了高效的几何渲染和灵活的拓扑拾取能力，适合用于工程可视化、模型审查及 CAD 数据分析。

---

## 核心功能

### 📁 文件导入

* 支持导入常见 CAD 文件格式：**STEP、IGES、BREP**。
* 通过 OCC 的读取接口实现装配结构解析。
* 支持拖拽文件导入和菜单导入方式。

### 🧩 模型显示

* 基于 **OpenSceneGraph** 的高性能三维渲染。
* 支持模型层级显示与装配树管理。
* 可对模型进行旋转、缩放、平移等视图操作。

### 🎯 拾取与选择

* 多种拾取模式：单击、矩形、套索、多边形、圆形、悬停。
* GPU 加速拾取，提升大模型选择性能。
* 支持拾取后的高亮显示与属性查询。

### 🧱 几何与拓扑管理

* 提供 **BodyManager** 管理几何体实例与颜色属性。
* 提供 **SelectManager** 管理拾取对象和选择状态。
* 支持对象层级组织与拓扑结构访问。

### 💾 导出功能

* 导出当前场景、选中对象或拾取形状。
* 可扩展导出格式，便于与其他 CAD 系统集成。

### 🪟 图形界面

* 使用 **Qt Widgets** 构建主窗口与菜单栏。
* OSG 通过 `osgQOpenGLWidget` 嵌入 Qt 实现渲染。
* 友好的交互界面与简洁的操作逻辑。

---

## 技术特性

* **跨平台支持**：Windows / Linux / macOS。
* **C++17** 实现，代码模块化，便于二次开发。
* 支持 **OSG + Qt** 融合渲染框架。
* 可扩展的拾取与显示模块设计，适合科研和工程项目集成。

---

SelOO 的目标是提供一个可扩展的、轻量高效的 CAD 模型查看与分析工具，使开发者能够在此基础上快速构建更复杂的工程可视化应用。
